# Changelog

## 1.0.0 - 2026-03-30

- Se estabilizó la estructura Maven del proyecto y su ejecución por consola.
- Se reorganizaron paquetes internos para separar CLI, concurrencia y recorridos.
- Se añadió contexto pedagógico por demo: objetivo, observación esperada y conclusión.
- Se incorporó el modo rápido `--fast` para clase, CI y revisión inicial.
- Se agregaron pruebas de humo para todas las demos registradas.
- La CI ahora valida el proyecto en JDK 17 y JDK 21.
- Se publicó una guía de estudio inicial para estudiantes.
